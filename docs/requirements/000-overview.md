# VBA for Excel Language Extension

# 要件定義書（完全版）

既存要件ドラフトを基礎とし、機能拡張を統合


---

## 0. 前提とゴール

### 0.1 前提（固定条件）

* 対象：**Excel VBA のみ**
* **Option Explicit を前提**
* **Win64 必須**

  * `PtrSafe` 必須
  * `LongPtr / LongLong` 優先サポート
* 編集支援専用（VBE実行環境・マクロ実行は対象外）

---

### 0.2 ゴール（理想ユーザー体験）

VS Code上で `.bas / .cls / .frm` を開いた際に、VBEを超える編集支援を提供する。

#### 提供体験

* リアルタイム構文診断（壊れたコードでも解析継続）
* 高精度IntelliSense
* 定義ジャンプ・参照検索・安全リネーム
* セマンティックハイライト
* VBAに最適化された型推論
* VBA特有仕様（Win64 / Option Explicit）への厳格対応
* Officeオブジェクトモデルを考慮した補完
* リファクタリング支援
* コード整形
* 高度静的解析

---

## 1. 対象範囲

### 1.1 入力ファイル

* `.bas` 標準モジュール
* `.cls` クラスモジュール
* `.frm` フォーム
* `.frx` フォームバイナリ

  * フォーム定義部は初期テキスト扱い
  * コード部を解析対象
  * `.frx` は配置オブジェクトを解析

---

### 1.2 VBA言語要素対応範囲

#### 最優先対応

* 宣言
* 手続き境界
* 名前解決に必要な構文

#### 対応構文群

* 宣言：`Option / Dim / Private / Public / Friend / Const / Enum / Type`
* 手続き：`Sub / Function / Property Get/Let/Set`
* Win64：`Declare PtrSafe`
* 制御構文：`If / For / Do / Select / With / On Error`
* 条件コンパイル
* 属性行
